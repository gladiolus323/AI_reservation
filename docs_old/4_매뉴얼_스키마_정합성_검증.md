## 1. 개요

### 1.1 검증 목적

병원 검사 예약 업무 매뉴얼(38페이지)에 기술된 **모든 규칙**이 설계된 스키마로 정형화 가능한지 검증하고, 정형화 불가능한 규칙에 대한 처리 방안을 제시합니다.

| 원본 매뉴얼 | 통합 검사예약업무 매뉴얼.pdf (38p) |
| 스키마 | EXAM_MASTER, EXAM_RELATION_RULES, EXAM_CONDITION_RULES |
| 추출된 규칙 | 131개 |

**구현 예시 (Python)**:
```python
def calculate_fasting_end(exam_datetime, fasting_hours):
    """검사시간 기준 금식 종료 시간 계산"""
    return exam_datetime - timedelta(hours=fasting_hours)

def calculate_holter_removal(attach_date, duration_days):
    """Holter 제거일 계산 (주말 제외)"""
    removal = attach_date + timedelta(days=duration_days)
    while removal.weekday() >= 5:  # 토/일
        removal += timedelta(days=1)
    return removal
```
