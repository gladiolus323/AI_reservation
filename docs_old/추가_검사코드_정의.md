# 추가 검사코드 정의

**작성일**: 2025-12-22
**수정일**: 2025-12-29

---

## 개요

constraints.csv(251개)에 없지만 매뉴얼 규칙에서 필요한 검사들을 새로 정의합니다.

**총 17개 검사 추가** (기존 14개 + 신규 3개)

---

## 코드 체계

기존 constraints.csv 코드 체계를 따름:
- **RC**: CT (Radiology CT)
- **RM**: MRI (Radiology MRI)
- **RF**: Fluoroscopy (투시촬영)
- **RU**: Ultrasound (초음파)
- **RX**: X-ray/기타 영상
- **NM**: Nuclear Medicine (핵의학)
- **SC**: Special Check (특수검사)

## 상세 정의

### RX010001 - BMD
- **검사명**: Bone Mineral Density (골밀도검사)
- **장비**: DEXA (Dual-energy X-ray Absorptiometry)
- **소요시간**: 15분
- **관련 규칙**: CT(조영제) 후 3일 간격 필요

### RX020001 - Mammography
- **검사명**: Mammography (유방촬영술)
- **장비**: Mammography
- **소요시간**: 20분
- **관련 규칙**: Mammography → 핵의학 순서 필수

### RX030001 - EOS
- **검사명**: EOS (전척추촬영)
- **장비**: EOS Imaging System
- **소요시간**: 15분
- **관련 규칙**: EOS → Bone scan 순서 필수

### RF020001 - Defecogram
- **검사명**: Defecogram (배변조영술)
- **장비**: Fluoroscopy
- **소요시간**: 30분
- **관련 규칙**: 대장내시경 후 1일 간격, Manometry 후 3일 간격

### RF030001 - Esophagography
- **검사명**: Esophagography (식도조영술)
- **장비**: Fluoroscopy
- **소요시간**: 20분
- **관련 규칙**: CT 후 1일 간격, Esophagography 후 CT 7일 간격

### RF040001 - VCUG
- **검사명**: VCUG (배뇨방광요도조영술)
- **장비**: Fluoroscopy
- **소요시간**: 30분
- **관련 규칙**: DMSA Renal Scan 후 2일 간격

### SC040001 - UBT
- **검사명**: UBT (Urea Breath Test, 요소호기검사)
- **장비**: Breath Test Analyzer
- **소요시간**: 20분
- **관련 규칙**: UBT → 위내시경 60분 간격 (순서 필수)

### SC050001 - Esophageal Manometry
- **검사명**: Esophageal Manometry (식도내압검사)
- **장비**: Manometry System
- **소요시간**: 40분
- **관련 규칙**: 식도내압 → 위내시경 30분 간격 (순서 필수)

### SC050002 - Anorectal Manometry
- **검사명**: Anorectal Manometry (항문내압검사)
- **장비**: Manometry System
- **소요시간**: 40분
- **관련 규칙**: 항문내압 → 대장내시경 1일 간격

### SC060001 - CPX
- **검사명**: CPX (Cardiopulmonary Exercise Test, 심폐운동검사)
- **장비**: CPX System
- **소요시간**: 60분
- **관련 규칙**: Heart MR과 동일일 불가

### SC070001 - PSG
- **검사명**: PSG (Polysomnography, 수면다원검사)
- **장비**: PSG System
- **소요시간**: 480분 (야간)
- **관련 규칙**: MSLT 전날 시행

### SC070002 - MSLT
- **검사명**: MSLT (Multiple Sleep Latency Test, 주간졸림검사)
- **장비**: PSG System
- **소요시간**: 480분 (주간)
- **관련 규칙**: PSG 다음날 8:30 시행

### RM060001 - Breast MRI
- **검사명**: Breast MRI (유방 자기공명영상)
- **장비**: MRI
- **소요시간**: 40분
- **관련 규칙**: 유방 초음파와 다른 날 시행 시 2일 간격 필요

### RU060001 - Breast Sono
- **검사명**: Breast Sono (유방 초음파)
- **장비**: Ultrasound
- **소요시간**: 30분
- **관련 규칙**: 유방 MRI 후 2일 간격 필요 (다른 날 시행 시)

### RM070002 - Heart MRI
- **검사명**: Heart MRI (심장 MRI, Cardiac MRI)
- **장비**: MRI
- **소요시간**: 120분 (2시간)
- **금식**: 8시간
- **조영제**: Y
- **운영시간**: 08:00~17:00
- **관련 규칙**:
  - Cardiac CT 후 Heart MRI 순서 필수 (금식검사 순서)
  - CPX와 동일일 불가
  - Stress Echo와 동일일 불가

### SC060002 - TMT
- **검사명**: TMT (Treadmill Test, 운동부하검사)
- **장비**: Treadmill
- **소요시간**: 45분
- **관련 규칙**: Coronary CT와 동일일 불가

### SC060003 - Stress Echo
- **검사명**: Stress Echo (부하심초음파)
- **장비**: Echo + Treadmill/Dobutamine
- **소요시간**: 60분
- **관련 규칙**: Heart MRI와 동일일 불가
